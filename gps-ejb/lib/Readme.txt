Configuration d'envoi d'un SMS avec modem :

Dans le dossier jar sous le dossier lib du projet gps-ejb, ajouter comm.jar, smslib-3.3.3.jar et smsserver-3.5.4.jar
à votre projet.

Dans le dossier jdk sous le dossier lib du projet gps-ejb, ajouter comm.jar et javax.comm.properties dans le dossier lib
de votre JDK (Java)
ajouter win32com.dll dans le dossier bin de votre JDK

ps: JDK version 32 bits !!
